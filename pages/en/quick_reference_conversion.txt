===== Extrusions =====

The 'extrusions' are accessed through the modeling API using the following:

<code javascript>
const { extrudeLinear, extrudeRegtangular, extrudeRotate  } = require('@jscad/modeling').extrusions
</code>

^  Extrusion of 2D Shapes  ^  Notes  ^
| const newshape = extrudeLinear({height: 20, twistAngle: Math.PI, twistSteps: 20}, oldshape))  | [[https://www.jscad.xyz/docs/module-modeling_transforms.html#.extrudeLinear | API]]  |
| const newshape = extrudeRectangular({size: 3, height: 15}, oldshape)  | [[https://www.jscad.xyz/docs/module-modeling_transforms.html#.extrudeRectangular | API]]  |
